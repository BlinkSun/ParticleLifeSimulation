# ParticleLifeSimulation
A C# Winforms port of [Brainxyz's Artificial Life](https://www.youtube.com/watch?v=0Kx4Y9TVMGg).

> *A simple program to simulate primitive Artificial Life using simple rules of
> attraction or repulsion among atom-like particles, producing complex
> self-organzing life-like patterns.*
> 
>
> – from the [original repository](https://github.com/hunar4321/life_code)

## Running the App

1. Install [Visual Studio](https://visualstudio.microsoft.com/) if not already installed, and be sure to include the dependencies for .NET and desktop development.
    - _Note: You need to use Visual Studio (not Visual Studio Code). Visual Studio Code does not have winforms designers._
2. [Download]() this repository and unzip the contents.
3. Open the `ParticleLifeSimulation.sln` file in Visual Studio.
